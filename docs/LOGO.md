## Gold bar — standard for all MMG apps

Every MMG app icon and header must include a gold bar (`#B8973C`) beneath the logo mark. This is the visual thread connecting all MMG tools to the main brand.

### App icons (PNG — the correct method)

Icons are generated by the PowerShell script:
```
C:\Users\andyg\Desktop\cursor\mmg-tools\scripts\generate-pwa-icons.ps1
```

To add a new app: add an entry to the `$apps` array in that script with `Slug`, `Label`, `Logo`, `Background`, `Foreground`, and `Accent` (`#B8973C`). Then run:
```powershell
cd C:\Users\andyg\Desktop\cursor\mmg-tools\scripts
powershell -ExecutionPolicy Bypass -File generate-pwa-icons.ps1
```

Copy the generated `icons/` folder from `mmg-tools/<slug>/icons/` into the new app's `public/icons/` and wire up in `layout.jsx` (see below).

The script produces `icon-192.png`, `icon-512.png`, and `apple-touch-icon.png` — all with the real MMG golf ball logo, correct font (Cormorant Infant Bold), and gold bar at bottom.

### In the app header (HTML/CSS)

A `<span>` immediately after the `<img>`, styled as a full-width 3px gold bar with a small top margin:

```jsx
<div className={styles.cornerBrand}>
  <img className={styles.heroLogo} src="/MMG_Logo_Gold_Transparent.png" alt="Mr Mallorca Golf" />
  <span className={styles.logoGoldBar}></span>
</div>
```

```css
.logoGoldBar {
  display: block;
  width: 100%;
  height: 3px;
  background: var(--gold); /* #B8973C */
  border-radius: 2px;
  margin-top: 6px;
}
```

Do not skip this. It should appear on every MMG tool, tracker, or internal app — not just the main site.

## This project (Strokes Gained)

This app uses **`public/MMG_Logo_Gold_Transparent.png`** — a copy of the black transparent logo rendered gold via CSS `filter` in `shot-tracker.module.css`.

The filter stack in `.heroLogo`:
```css
filter:
  invert(63%) sepia(42%) saturate(608%) hue-rotate(4deg) brightness(91%) contrast(90%)
  drop-shadow(0 16px 26px rgba(44, 42, 39, 0.08));
```
This converts the black logo to `#B8973C` (MMG gold). Adjust `brightness()` to go lighter or darker. Filter values were computed for that exact hex — use [cssfiltertools.com](https://www.cssfiltertools.com/filter-to-color/) to generate values for any other colour.

## Adding a new colour variant

1. Copy `MMG_Logo_Black_Transparent.png` from the MMG site repo into this project's `public/` folder with a descriptive name (e.g. `MMG_Logo_Slate_Transparent.png`).
2. Apply a CSS `filter` chain to tint it. Use a tool like [CSS Filter Generator](https://www.cssfiltertools.com/filter-to-color/) to find the right filter for any hex colour.
3. Reference the new file in the `<img src="...">` and apply the filter class.
