# Mr Mallorca Golf Shot Tracker

Standalone Mallorca-first golf tracker for phone-based GPS shot tracking, custom hole maps, round stats, handicap history, and strokes-gained-style feedback.

## Local use

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```

## On-course use

Deploy to Netlify or Vercel, then open the HTTPS URL on your phone. Browser GPS requires HTTPS.
